# KCS templates

Use these templates when adding or updating KCS notes.

## Available templates

- [kcs-template-contract.md](kcs-template-contract.md) — for ownership/contracts and integration boundaries.
- [kcs-template-troubleshooting.md](kcs-template-troubleshooting.md) — for regression triage pages.
- [kcs-template-runbook.md](kcs-template-runbook.md) — for contributor execution workflows.
- [kcs-template-matrix.md](kcs-template-matrix.md) — for producer/consumer ownership matrices.
- [kcs-template-reference.md](kcs-template-reference.md) — for compact reference/decision pages.

## Minimum quality bar

Every KCS note should include:

- symptom,
- operational context,
- decision rules/contracts,
- file-path anchors,
- failure modes,
- test anchors,
- discoverability links.
